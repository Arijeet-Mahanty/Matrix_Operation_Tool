# Matrix Operation Tool: A NumPy-Powered Matrix Tool
A simple and efficient Python script for performing core matrix operations. This project leverages the power of the NumPy library to handle mathematical computations between two matrices with ease.
# 📌 Project Status
This is a foundational version of the tool. I am currently using it to implement basic operations and plan to upgrade it with more complex linear algebra features and a more interactive interface in the future.
# 🛠️ Features (Current)
The tool currently supports the following operations between two matrices:

Matrix Addition: Element-wise addition.

Matrix Subtraction: Element-wise subtraction.

Matrix Multiplication: Standard dot product multiplication.
# 🚀 Technologies Used
Python 

NumPy: The primary library for high-performance numerical calculations.

# 📂 Repository Structure
Simple_Matrix_Tool.py: The main Python script containing the logic for matrix math.

README.md: Project documentation and roadmap.

# 🔧 Setup and Usage
Clone the Repo: git clone https://github.com/Arijeet-Mahanty/Matrix_Operation_Tool.git

Install NumPy:
If you don't have NumPy installed, run:  pip install numpy
Run the Script: python Simple_Matrix_Tool.py
